Tushare Pro 可下载数据权限

这页只负责讲清楚 Tushare Pro 能下什么、限频怎么理解、3 个 token 怎么轮换使用。TARDIS 仍然是站内量化数据主入口。

TARDIS 仍然优先：如果你的核心需求是高频、盘口、期权、现货与交易所历史数据，优先看 TARDIS 数据指南。 Tushare 更适合拿 A 股、新闻、全球指数和跨市场补充数据，不喧宾夺主。

一、可下载数据范围

A 股与舆情

  • 互动易问答（沪深）
  • 筹码分布
  • 融资融券基础数据
  • 新闻数据

分钟与衍生品

  • 股票分钟线
  • 期货数据
  • 期权数据
  • 主连 / 连续合约数据

跨市场

  • 港股数据
  • 美股数据
  • 外汇数据
  • 全球指数数据
  • 指数数据

二、补充说明

  • 主连分钟直下：不算直接支持，通常需要先做 `mapping`，再映射到实际合约分钟数据。
  • 分钟数据单个 API 最多每分钟下两次，不能按无限制高频方式去打。
  • 期货数据支持下载历史期货分钟数据，不支持实时期货分钟数据。
  • 发货 3 个 token，可轮换使用，能够缓解部分接口的每分钟或每日限频。

三、更适合什么场景

Tushare 更适合做 A 股研究、事件驱动、舆情辅助、港美股补充、全球指数同步和中低频研究所需的数据补全。它不是 TARDIS 的替代品，而是量化数据栈里的另一层。

四、联系与交付

当前交付方式包含 3 个 token，可轮换使用。优先点击闲鱼链接直接打开：闲鱼商品页；如果闲鱼链接失效，再联系微信 dreamsjtuai。
